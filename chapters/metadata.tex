\infoabstract{%
本设计面向同济大学 Unix V6++ 教学操作系统，在团队整体 Rust 重构和 RISC-V 迁移工作的基础上，重点完成内存管理、文件系统及相关 I/O 路径的分析与实现。设计保留 Unix V6 风格的页式管理、inode、目录项、缓冲区缓存和多级块映射机制，引入类型化地址、Buddy 与 Slab 分层分配、RAII 资源回收、引用封装和显式错误处理，并适配 Sv39 分页与 VirtIO 块设备。测试表明，Rust IA32 与 Rust RISC-V 版本均能完成文件系统装载、shell 运行和文件读写等基本任务。
}
