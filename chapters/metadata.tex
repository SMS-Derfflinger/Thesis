\infoabstract{%
本课题面向同济大学 Unix V6++ 操作系统教学平台的现代化改造，分析原 IA32 与 C++ 实现中内存管理、文件系统、缓冲区缓存和设备管理等核心模块的结构与局限，使用 Rust 对内存管理和文件系统相关模块进行重构，并探索向 RISC-V 64 位架构迁移。重构过程中保留 Unix V6++ 的教学结构和经典机制，引入 Buddy 与 Slab 分层分配、类型化地址抽象、引用封装和显式错误处理，以提升安全性、可维护性和模块边界清晰度。系统同时适配串口、块设备和 QEMU 虚拟化环境，并通过功能测试与性能测试对比不同实现方案，为教学操作系统的持续演进提供参考。
}
